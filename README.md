# Sistem Pengaduan Masyarakat

Project website sistem pengaduan masyarakat berbasis web.

## Deskripsi
Sistem ini dibuat untuk mempermudah masyarakat dalam menyampaikan laporan atau pengaduan secara online serta membantu admin dalam mengelola laporan.

## Fitur
- Registrasi dan login pengguna
- Mengirim pengaduan
- Upload bukti laporan
- Pengelolaan status pengaduan
- Dashboard admin

## Teknologi
- HTML
- CSS
- PHP
- MySQL

## Pengembangan
Project ini dikembangkan sebagai project pembelajaran Teknik Informatika.
